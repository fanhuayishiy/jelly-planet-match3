# 果冻星球消消乐

一个单文件 H5 三消游戏，使用原生 HTML、CSS 和 JavaScript 实现。项目包含关卡目标、步数限制、连锁消除、特殊果冻、粒子特效和响应式界面，可直接双击运行，也可以部署到 GitHub Pages。

## 在线运行

在线试玩：

https://fanhuayishiy.github.io/jelly-planet-match3/

本地打开：

```bash
start match3.html
```

或使用任意静态服务器：

```bash
npx serve .
```

然后访问 `match3.html`。

## 玩法特性

- 8 x 8 三消棋盘
- 游戏开始页面，包含开始、继续和玩法速览
- 关卡地图和游戏内快速关卡条，可直接跳到后续关卡
- 拖动水果到相邻格子交换
- 无效交换会自动回弹
- 消除后自动掉落和补齐
- 连锁消除直到棋盘稳定
- 20 个关卡，包含目标分、收集目标、冰晶目标和步数限制
- 第 6 关后加入固定冰晶格，冰晶不能交换、不能拖动，也不会被掉落或洗牌移动；第一次命中先打碎冰壳并计入目标
- 4 连生成横向或纵向火箭
- 5 连生成彩虹糖，T/L 型交叉消除生成爆裂糖
- 火箭、爆裂糖、彩虹糖移动到相邻格子即可触发，双彩虹糖可全屏清除
- 提示系统会高亮可交换步骤
- 无可走步时自动洗牌，避免棋盘死局
- 锤子、洗牌、彩虹糖三类关卡道具
- 粒子爆炸、星芒、冲击波、浮动加分、棋盘脉冲和弹窗反馈
- Combo 大字、得分跳动、屏幕震动、触觉震动和 Web Audio 音效
- 兼容桌面鼠标与移动端触控

## 部署到 GitHub Pages

推送到 GitHub 后，在仓库设置中打开 Pages：

1. 进入仓库的 `Settings`
2. 打开 `Pages`
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/root`
5. 保存后访问 GitHub Pages 给出的地址
